Le projet

Ce site constitue le projet Developpement Web effectué dans le cadre d’un projet scolaire à l’ISEN Nantes pour les 3e années. Il s’agit d’une étude d’une base de données de 73643 lignes et 21 colonnes représentant les accidents de la route en France, en 2009. Il se situe dans la continuité de deux autres projets (Big Data et Intelligence Artificielle) et permet la modélisation et la manipulation d'une base de données via une interface web. Il permet aussi l'affichage des données traitées lors des précedents projets sur une page web.
